# TellerMachine
Automated teller machine simulator (JetBrains Academy's project)

Simulator supports the following operations:
<pre>
1. Create an account
2. Log into account
   1. Balance
   2. Add income
   3. Do transfer
   4. Close account
   5. Log out
   6. Exit
3. Exit
<pre>

Data is stored in SQLite database.

It should be started from terminal by: python banking.py
